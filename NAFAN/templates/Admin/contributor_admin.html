Contributer Admin

Repositories

A Contributor Admin has access to repositories as assigned by a NAFAN Admin. Contributor Admins can have access to multiple repositories. A Contributer Admin can edit the repository information and split and merge repositories assigned. I am not sure if the Contributor Admin should have the ability to activate and deactivate a repository. They probably should.

Users

A Contributor Admin can perform all user actions. These include adding users, editing user information and activating and deactivating a user. The Contributor Admin also designates which repositories their users can access.

Audit

A Contributor Admin has access to audit information concerning their repositories.
